# Cursor Website Clone (HTML & CSS)

This project is a **static HTML & CSS recreation** of the Mintlify website UI.  
The goal was to closely match the original layout, typography, colors, and overall visual feel using **pure HTML and CSS**.

## 🎨 Fonts Used

-   **Inter** (Primary Font)
-   **Geist Mono** (Used for labels and metadata text)

Both fonts are imported and configured inside `style.css`.

------------------------------------------------------------------------

## 🎨 Color System

The project uses CSS root variables for consistent design.

Key colors:

-   Main Background: Dark LAB-based color
-   Section Background: `#141516`
-   Accent/Brand Color: Green LAB-based tone
-   Primary Text: White
-   Secondary Text: Semi-transparent white
-   Soft Borders: Light translucent borders
